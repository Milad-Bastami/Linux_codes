shell wild cards and expansion

- astrisk  matches zero or more characters (ignores hidden files)
- ? matches one characters (ignore hidden files)
- [A-Z]  alphanumeric range
- [0-9]  alphanumeric range

examples:

ls zmays[AB]_R1.fastq
ls zmays*
ls zmays?_R1.fastq

brace expansion 'text-{1,2,3}'

expand the comma seperated values in curle braces. There should be o space between values. The -p argument let mkdir to create any subdirecory (here data) if needed. brace expansion will exand irrespective o wether file or directory exist. but wildcards expand to existing files. mkdir -p zmays/{data/seqs,analysis/scripts}
